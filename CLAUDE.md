# AR22 LCU 펌웨어 — 작업 규칙 (Working Rules)

## 배경 (왜 이 규칙이 있는가)

이 폴더의 펌웨어는 **이전 제품에서 정상 동작이 검증된 코드**다. 이번에 **업그레이드 제품에
재사용**하기 위해, 검증된 기존 펌웨어를 **최대한 그대로 보존**하는 것이 목표다.
따라서 모든 작업의 기본 원칙은 다음과 같다:

- 검증된 기존 동작/로직을 임의로 바꾸지 않는다.
- 변경은 꼭 필요한 부분만, 최소·국소적으로 한다.
- 업그레이드를 위한 신규 기능은 가능하면 기존 코드를 건드리지 않고 분리해 추가한다.
- "개선"으로 보이는 리팩터링·정리도 동작에 영향이 있으면 함부로 하지 않는다.

## ⚠️ 최우선 규칙: 시스템 동작 변경 전 반드시 사용자 확인

이 프로젝트는 방사선 검출기(AR22 LCU)의 실동작 펌웨어입니다.
**현 시점 시스템의 동작을 바꾸는 모든 변경은 실행 전에 반드시 사용자(obt.sup@gmail.com)의 명시적 확인을 받는다.**

확인 절차:
1. 무엇을, 왜 바꾸려는지 먼저 설명한다.
2. 변경이 동작에 미치는 영향(부작용·위험)을 함께 제시한다.
3. 사용자가 명시적으로 승인한 **후에만** 실제 수정을 진행한다.
4. 한 번의 승인은 그 건에만 적용된다. 다른 변경에는 다시 확인을 받는다.

> 한 작업 안에서 추가 변경이 필요해지면, 임의로 진행하지 말고 멈춰서 다시 확인을 받는다.

---

### 확인이 **필요한** 변경 (동작 변경으로 간주)
- 펌웨어 런타임 로직 / 제어 흐름 / 알고리즘 수정
- IO·핀 설정, GPIO 모드, 주변장치(SPI/I2C/UART/TIM/DMA) 설정 변경
- 핀 매핑·라벨, 인터럽트/NVIC, 클럭·타이밍 변경
- 기능 추가·삭제·비활성화 (예: 핀 제거, 드라이버 교체)
- `.ioc` 수정, 드라이버 교체/삭제, `stm32f4xx_hal_conf.h` 모듈 활성화
- 보정계수·임계값·기본값 등 동작에 영향을 주는 상수 변경

### 확인 **없이** 진행 가능한 작업 (동작 불변)
- 읽기 전용 분석·검색·코드 리뷰
- 빌드/컴파일/정적 검증 (산출물 생성 포함)
- 질문 답변, 설명, 문서 작성
- 동작에 영향이 없는 순수 주석/포매팅 (단, 애매하면 먼저 물어본다)

**판단이 애매하면 "동작 변경"으로 간주하고 먼저 확인을 받는다.**

## 코드 주석 규칙 (필수)

**작성하거나 수정하는 모든 코드에는 반드시 주석을 단다.**

- 새 함수/블록: 무엇을·왜 하는지(동작 의도, 단위, 값의 범위, 부작용)를 주석으로 남긴다.
- 기존 코드 수정: 변경한 부분의 의도가 드러나도록 주석을 함께 남긴다.
- 비자명한 상수·매직넘버·하드웨어 의존값(레지스터 값, 주소, 비트, 타이밍, 핀)은 의미를 주석으로 명시한다.
- 주석 언어·스타일은 주변 코드에 맞춘다.
- 주석만 추가/수정하는 것은 동작 불변이므로 사전 확인 없이 진행 가능하다.
- 단, 검증된 기존 코드 전체를 주석만을 위해 대량 수정하지는 않는다. **손대는(추가·수정하는) 범위에 한해** 주석을 필수로 보강한다.

## 참고: 빌드 방법
STM32CubeIDE 미설치 환경이며, ARM GNU Toolchain으로 직접 빌드/검증한다.
- 코어: Cortex-M4, FPU `fpv4-sp-d16` (hard), `-mthumb`
- 정의: `STM32F446xx`, `USE_HAL_DRIVER`
- 링커: `STM32F446VETX_FLASH.ld`, `--specs=nano.specs --specs=nosys.specs -u _printf_float`
- 산출물: `build/` 폴더 (.elf/.hex/.bin/.map)
- 빌드는 동작을 바꾸지 않으므로 사전 확인 없이 수행 가능.

## ⚠️ 참고: CubeMX 재생성 주의
`.ioc`를 재생성하면 `main()` 초기화 블록에 `MX_IWDG_Init()`가 다시 삽입되어 **POST 부트 루프**
("셀프테스트에서 멈춤")를 유발한다. 재생성 후 그 이른 호출을 삭제하고 **메인 루프 직전**의
호출만 남긴다. 그 외 USER CODE 커스터마이즈 보존 여부도 점검한다. 상세 체크리스트:
[`docs/CUBEMX_REGEN_NOTES.md`](docs/CUBEMX_REGEN_NOTES.md).
